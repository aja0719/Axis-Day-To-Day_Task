Collections Frameworks=>
        The Collection in Java is a framework that provides an architecture 
           to store and manipulate the group of objects.

         Java Collection framework provides many interfaces (Set, List, Queue, Deque) 
         and classes (ArrayList,Vector,LinkedList, PriorityQueue, HashSet, LinkedHashSet,TreeSet).

      Any Collection class holds only group of objects .
